This project presents an interactive Sales Data Dashboard built using Microsoft Power BI. The goal is to analyze sales performance, identify trends, and generate actionable insights for better business decision-making.

The dashboard provides a comprehensive view of sales metrics such as revenue, profit, customer segmentation, and product performance.

~ Objectives
Analyze overall sales performance
Identify top-performing products and regions
Track profit trends and revenue growth
Understand customer purchasing behavior
Enable data-driven decision-making
~ Tools & Technologies
Power BI Desktop (.pbix)
Data Modeling
DAX (Data Analysis Expressions)
Data Visualization
~ Dataset Description

The dataset used in this project includes:

Sales transactions
Product details
Customer information
Regional data
Order and shipping details


📊 Dashboard Features
🔹 Key KPIs
Total Sales

Total Orders
